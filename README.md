### 협업을 위한 커밋 규칙

#### Commit
* FEAT : 새로운 기능 추가
* FIX : 버그 수정
* DOCS : 문서 수정

#### Branch
* Main : 메인 브랜치
* Hotfix : 디버깅 브랜치 (버그 수정)
* Develop : 개발 브랜치
